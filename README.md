# Pivotal CMD

Command line tool for pivotal

## Installation

```bash
gem install pivotal-cmd
$ pivotal-cmd <command>
```

## Usage

First of all, create a __.pivotalcmdrc__ with the api token and project id
in your home directory (~/).

### Example:

```yaml
token: ee59...cssb
project_id: 999
```

###Available commands
- current

```bash
pivotal-cmd current
```
